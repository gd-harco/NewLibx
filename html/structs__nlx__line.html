Detailed Description

Structure to store a line and its properties.

Parameters
startThe start point of the line. Must be provided.
endThe end point of the line. Must be provided.
delta_xThe difference between the x coordinates of the start and end points. Calculated automatically.
delta_yThe difference between the y coordinates of the start and end points. Calculated automatically.
var_yThe variation of the y coordinate. Used when drawing the line. Determined automatically.
var_xThe variation of the x coordinate. Used when drawing the line. Determined automatically.
